## Lua安装与测试

```sh
curl -R -O http://www.lua.org/ftp/lua-5.3.5.tar.gz
tar zxf lua-5.3.5.tar.gz
cd lua-5.3.5
# For Linux
make linux test
make linux install
# For Mac
make macosx test
make macosx install
lua -v
```
